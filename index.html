판매가 계산기

운임 균등배분 공식 (수량 기준):
① 구입단가(원) = 구입단가(USD) × 환율   ② 배분운임단가 = 총 운임 ÷ 총수량
③ 각 품목 판매단가 = (구입단가(원) / (1 - 마진율)) + 배분운임단가
④ 판매이익단가 = 판매단가 - 구입단가 - 배분운임단가   ⑤ 총판매이익 = 판매이익단가 × 수량

운임 비례배분 공식 (원가 기준):
① 품목 원가합(원) = 구입단가(원) × 수량   ② 전체 원가합 = 모든 품목 원가합의 합
③ 품목 배분운임(총액) = 총 운임 × (품목 원가합 / 전체 원가합)
④ 배분운임단가 = 품목 배분운임(총액) / 수량
⑤ 판매단가 = (구입단가(원) / (1 - 마진율)) + 배분운임단가

예시 (환율 1,300원/USD): 품목A 2개($500), 품목B 3개($800), 총운임 100,000원
• 균등배분: 배분운임단가 = 10만 ÷ 5 = 2만원 (수량 5개)
• 비례배분: A원가합=130만, B원가합=312만 → A배분운임 = 10만×(130/442) ≈ 29,412원
