판매가 계산기

운임 균등배분 공식:
① 개당 운임 = 총 운임 ÷ 총수량
② 각 품목 배분 운임 = 개당 운임 × 해당 품목 수량
③ 각 품목 판매단가 = (원가 / (1 - 마진율)) + 개당 운임
④ 총 판매가 = 판매단가 × 수량

예시: 품목A 2개, 품목B 3개 → 총수량=5개, 총 운임 10만원
• 개당 운임 = 10만 ÷ 5 = 2만원
• 품목A 배분 운임 = 2만 × 2 = 4만원, 품목B 배분 운임 = 2만 × 3 = 6만원
• 품목A: 원가 100만원, 마진율 20% → 판매단가 = 100만/0.8 + 2만 = 127만원, 총 판매가 = 127만 × 2 = 254만원
