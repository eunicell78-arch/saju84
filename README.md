# 🔮 사주팔자 만세력 계산기 with AI

OpenAI ChatGPT를 활용한 전문 사주팔자 만세력 계산 및 AI 풀이 서비스

**천을귀인 만세력 v5.05 기준으로 정확한 계산**

## ✨ 주요 기능

### 1. 사주팔자 계산 ✅
- 생년월일시를 입력하면 사주팔자(四柱八字) 자동 계산
- 천간(天干)과 지지(地支) 정확한 계산
- 연주, 월주, 일주, 시주 구분 표시
- 한자 표기 지원
- **60갑자 순환 정확도**: 천을귀인 만세력과 100% 일치

### 2. 십신(十神) 분석 🆕
일간(日干)을 기준으로 다른 천간과의 관계 분석:
- **비겁(比劫)**: 비견, 겁재
- **식상(食傷)**: 식신, 상관
- **재성(財星)**: 편재, 정재
- **관살(官殺)**: 편관, 정관
- **인성(印星)**: 편인, 정인

### 3. 12운성(十二運星) 🆕
일간이 12지지에서 받는 기운 분석:
- 장생, 목욕, 관대, 건록, 제왕, 쇠, 병, 사, 묘, 절, 태, 양

### 4. 신살(神殺) 분석 🆕
- **천을귀인(天乙貴人)**: 귀인의 도움
- **역마살(驛馬殺)**: 이동과 변화
- **도화살(桃花殺)**: 인기와 이성운
- **공망(空亡)**: 빈 공간
- **원진(怨嗔)**: 원한과 불화

### 5. 형충회합(刑沖會合) 🆕
지지 간의 상호 작용 분석:
- **충(沖)**: 정면 대립
- **육합(六合)**: 두 지지의 조화
- **삼합(三合)**: 세 지지가 모여 오행 형성
- **방합(方合)**: 같은 방위의 조화
- **형(刑)**: 상호 해침

### 6. 대운(大運) 계산 🆕
10년 단위의 큰 운세:
- **순행/역행** 자동 판단 (양남음녀/음남양녀)
- **대운수** 계산 (3일 = 1년)
- **10개 대운 주기** 표시
- 각 대운의 십신, 12운성 분석

### 7. 세운(歲運) 계산 🆕
매년의 운세:
- 과거 5년, 현재, 미래 10년 간지
- 나이별 운세 흐름
- 현재 년도 강조 표시

### 8. 납음오행(納音五行) 🆕
60갑자별 특수 오행:
- 해중금(海中金), 노중화(爐中火) 등
- 30가지 납음오행 분류

### 9. 오행(五行) 분석
- 천간과 지지의 오행 분석
- 목(木), 화(火), 토(土), 금(金), 수(水) 개수 통계
- 오행의 균형 상태 확인

### 10. AI 사주 풀이 🤖 (강화됨)
OpenAI GPT-4를 활용한 전문적이고 종합적인 사주 해석:
- **사주 구조 분석**: 일간 강약과 사주 전체 평가
- **용신 선정**: 필요한 용신과 희신 분석
- **십신으로 본 성격**: 십신 배치로 성격과 직업 적성 분석
- **신살의 길흉**: 주요 신살의 의미와 영향
- **형충회합의 영향**: 상호작용이 삶에 미치는 영향
- **현재 대운/세운 해석**: 현재 운세 분석
- **향후 10년 운세**: 대운과 세운 흐름 전망
- **개선 방향 조언**: 오행 조화를 위한 실천적 조언

### 11. 결과 다운로드
- 사주팔자 및 AI 풀이 결과를 텍스트 파일로 다운로드

## 🚀 빠른 시작

### 로컬 실행

1. **저장소 클론**
```bash
git clone https://github.com/eunicell78-arch/saju84.git
cd saju84
```

2. **의존성 설치**
```bash
pip install -r requirements.txt
```

3. **OpenAI API 키 설정**

`.streamlit/secrets.toml` 파일 생성:
```bash
mkdir -p .streamlit
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
```

`.streamlit/secrets.toml` 파일 편집:
```toml
OPENAI_API_KEY = "sk-your-actual-api-key-here"
```

> OpenAI API 키는 [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys)에서 발급받을 수 있습니다.

4. **앱 실행**
```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501`로 접속하세요.


## 📚 사주팔자 개념

### 사주팔자란?
사주팔자(四柱八字)는 한 사람이 태어난 년(年), 월(月), 일(日), 시(時)를 천간(天干)과 지지(地支)로 표현한 것입니다.

- **천간(10개)**: 갑(甲), 을(乙), 병(丙), 정(丁), 무(戊), 기(己), 경(庚), 신(辛), 임(壬), 계(癸)
- **지지(12개)**: 자(子), 축(丑), 인(寅), 묘(卯), 진(辰), 사(巳), 오(午), 미(未), 신(申), 유(酉), 술(戌), 해(亥)

### 오행(五行)
- **목(木)**: 나무, 성장
- **화(火)**: 불, 열정
- **토(土)**: 흙, 안정
- **금(金)**: 쇠, 강인함
- **수(水)**: 물, 지혜
