# 🔮 만세력 - 사주팔자 계산기

동양의 전통 만세력을 웹에서 확인하세요!

## 기능
- ✅ 양력/음력 변환 (1900-2100년)
- ✅ 사주팔자 계산
- ✅ 오행 분석
- ✅ 음양 분석

## 사용 방법
1. 양력/음력 선택
2. 생년월일시 입력
3. 계산 버튼 클릭!

## 로컬 실행 방법
```bash
# 필요한 패키지 설치
pip install -r requirements.txt

# 앱 실행
streamlit run app.py
```

## 배포
Streamlit Cloud에서 무료 배포 가능

1. https://share.streamlit.io/ 접속
2. GitHub 로그인
3. "New app" 클릭
4. Repository: `eunicell78-arch/saju84` 선택
5. Branch: `main`
6. Main file: `app.py`
7. Deploy 클릭!
