# 사주팔자 만세력 계산기 (Four Pillars of Destiny Calculator)

한국 전통 명리학의 사주팔자를 계산하는 웹 기반 계산기입니다. 코딩 지식이 없는 일반인도 쉽게 사용할 수 있도록 직관적인 인터페이스로 설계되었습니다.

🔗 **Live Demo**: [GitHub Pages에서 사용하기](https://eunicell78-arch.github.io/saju84/)

## ✨ 주요 기능

### 1. 달력 시스템
- ✅ **양력/음력 선택 가능**: 출생일을 양력 또는 음력으로 입력
- ✅ **윤달 지원**: 음력 윤달 출생자 정확히 계산
- ✅ **양력↔음력 자동 변환**: 한국천문연구원 기준 (1900-2100년)

### 2. 사주팔자 계산
정확한 명리학 알고리즘을 사용하여 다음을 계산합니다:

- **연주 (年柱, Year Pillar)**: 입춘(立春) 기준으로 연도 결정
- **월주 (月柱, Month Pillar)**: 24절기 기준으로 월 결정
- **일주 (日柱, Day Pillar)**: 60갑자 순환 기준
- **시주 (時柱, Hour Pillar)**: 12시진 기준 (자시~해시)

### 3. 표시 정보
- 천간지지(天干地支) 한자 및 한글 표기
- 오행(五行) 정보: 목화토금수
- 십이지 동물띠
- 각 기둥의 의미 설명

## 🚀 사용 방법

### 기본 사용법

1. **달력 유형 선택**
   - 출생일이 양력이면 '양력' 선택
   - 출생일이 음력이면 '음력' 선택

2. **생년월일 입력**
   - 년: 1900~2100 사이의 값
   - 월: 1~12
   - 일: 1~31
   - 음력 윤달 출생인 경우 '윤달' 체크박스 선택

3. **출생 시간 입력**
   - 24시간 형식으로 입력 (예: 오전 5시 30분 → 5시 30분)
   - 분 단위까지 정확히 입력 가능

4. **계산하기 버튼 클릭**
   - 사주팔자 결과 자동 표시
   - 양력↔음력 변환 정보도 함께 표시

### ⚠️ 주의사항

#### 자시(子時) 처리
- **23:00~24:00**: 다음 날의 자시(子時)로 계산됩니다
- 예: 1992년 10월 23일 23:30 → 10월 24일 자시로 계산

#### 입춘(立春) 기준
- 연주(年柱)는 양력 1월 1일이 아닌 **입춘을 기준**으로 바뀝니다
- 입춘 이전 출생자는 전년도 연주로 계산됩니다
- 예: 2024년 2월 3일 출생 → 2023년 연주 (입춘이 2월 4일인 경우)

## 🔬 계산 알고리즘

이 프로그램은 다음의 검증된 알고리즘을 참고하여 구현되었습니다:

### 참고 레포지토리
1. **[yhj1024/manseryeok](https://github.com/yhj1024/manseryeok)** - TypeScript 기반 만세력 계산
2. **[0ssw1/sajupy](https://github.com/0ssw1/sajupy)** - Python 기반 음력 변환 및 절기 계산

### 주요 알고리즘

#### 1. 음력 데이터
- 한국천문연구원(KASI) 기준 1900-2100년 음력 데이터
- 각 연도별 월별 일수 및 윤달 정보 포함

#### 2. 양력↔음력 변환
```
기준일: 1900년 1월 31일 = 음력 1900년 1월 1일
방법: 기준일로부터의 일수 차이 계산
```

#### 3. 24절기 계산
- 입춘, 경칩, 청명 등 24개 절기의 날짜 계산
- 월주(月柱) 결정에 사용
- 연주(年柱)의 입춘 기준 연도 조정

#### 4. 60갑자 순환
천간(天干) 10개 × 지지(地支) 12개 = 60갑자

**천간 (Heavenly Stems):**
甲(갑), 乙(을), 丙(병), 丁(정), 戊(무), 己(기), 庚(경), 辛(신), 壬(임), 癸(계)

**지지 (Earthly Branches):**
子(자), 丑(축), 寅(인), 卯(묘), 辰(진), 巳(사), 午(오), 未(미), 申(신), 酉(유), 戌(술), 亥(해)

#### 5. 시주(時柱) 계산
12시진 체계:
- 子時 (자시): 23:00~01:00
- 丑時 (축시): 01:00~03:00
- 寅時 (인시): 03:00~05:00
- 卯時 (묘시): 05:00~07:00
- 辰時 (진시): 07:00~09:00
- 巳時 (사시): 09:00~11:00
- 午時 (오시): 11:00~13:00
- 未時 (미시): 13:00~15:00
- 申時 (신시): 15:00~17:00
- 酉時 (유시): 17:00~19:00
- 戌時 (술시): 19:00~21:00
- 亥時 (해시): 21:00~23:00

## 🎯 사주팔자 해석 가이드

### 일간(日干) - Day Master
- 사주팔자 해석의 **핵심**
- 본인의 성격과 운명을 나타냄
- 일주의 천간(위쪽 한자)이 일간

### 사주 구성 요소
1. **연주(年柱)**: 조상, 초년운 (0-20세)
2. **월주(月柱)**: 부모, 청년운 (20-40세)
3. **일주(日柱)**: 본인/배우자, 중년운 (40-60세)
4. **시주(時柱)**: 자녀, 말년운 (60세 이후)

### 오행(五Elements)
- **木(목)**: 나무, 봄, 성장, 인자함
- **火(화)**: 불, 여름, 열정, 활동성
- **土(토)**: 흙, 환절기, 안정, 신중함
- **金(금)**: 쇠, 가을, 결단, 의리
- **水(수)**: 물, 겨울, 지혜, 유연성
