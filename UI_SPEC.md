# ============================================================================
# 입력 섹션
# ============================================================================

## 1. 사용자 입력
- 성별 선택: 라디오 버튼 (남/여)
- 생년월일: 날짜 선택기
- 출생 시간: 시간 선택기
- 계산 버튼: "🔮 사주팔자 계산하기"

# ============================================================================
# 출력 섹션
# ============================================================================

## 2. 사주팔자 4기둥 (가로 배치)
각 기둥마다:
- 기둥 이름 (시주, 일주, 월주, 연주)
- 한자 간지 (큰 글씨)
- 한글 표기
- 십신 (파란색 박스)
- 12운성 (초록색 박스)
- 납음오행 (작은 글씨)

예시:
```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│   시주(時柱)  │   일주(日柱)  │   월주(月柱)  │   연주(年柱)  │
│     戊申      │     丁未      │     丙子      │     己丑      │
│  무신(戊申)   │  정미(丁未)   │  병자(丙子)   │  기축(己丑)   │
│  [십신: 상관] │  [일간: 丁]  │  [십신: 겁재] │  [십신: 식신] │
│ [12운성: 목욕]│ [12운성: 관대]│ [12운성: 절] │ [12운성: 묘]  │
│ 납음:大驛土   │ 납음:天河水   │ 납음:澗下水   │ 납음:霹靂火   │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

## 3. 오행 분석
- 천간 오행 나열
- 지지 오행 나열
- 오행 개수 통계 (5개 칸으로 표시)
  ```
  목(木): 0개 | 화(火): 2개 | 토(土): 4개 | 금(金): 1개 | 수(水): 1개
  ```

## 4. 신살 (2열 배치)
좌측 열:
- 천을귀인 (초록색 - 길신)
- 역마살 (파란색 - 정보)
- 도화살 (주황색 - 경고)

우측 열:
- 공망 (빨간색 - 흉신)
- 원진 (주황색 - 경고)

예시:
```
✨ 신살 (神殺)
┌────────────────────────┬────────────────────────┐
│ 천을귀인: 천을귀인(子),  │ 공망: (없음)            │
│         천을귀인(申)    │                        │
│ 역마살: (없음)          │ 원진: 원진(子-未)       │
│ 도화살: 도화살(子)      │                        │
└────────────────────────┴────────────────────────┘
```

## 5. 형충회합 (3열 배치)
- 충(沖) - 빨간색
- 합(合) - 초록색 (육합, 삼합, 방합)
- 형(刑) - 주황색

예시:
```
⚡ 형충회합 (刑沖會合)
┌──────────────┬──────────────┬──────────────┐
│ 충(沖)        │ 합(合)        │ 형(刑)        │
│ 연지-일지 충   │ 육합:         │ 무례지형       │
│ (丑↔未)      │ 연지-월지 육합 │ (丑+未)       │
│              │ (丑+子)       │              │
│              │ 삼합:         │              │
│              │ 반합 水局      │              │
│              │ (子+申)       │              │
└──────────────┴──────────────┴──────────────┘
```


## 7. 세운 (4열 그리드)
과거 5년 + 현재 + 미래 10년 = 16개 항목

현재 년도는 초록색으로 강조:
```
📅 세운 (歲運)
┌──────────┬──────────┬──────────┬──────────┐
│ 2021년   │ 2022년   │ 2023년   │ 2024년   │
│ 辛丑     │ 壬寅     │ 癸卯     │ 甲辰     │
│ (13세)   │ (14세)   │ (15세)   │ (16세)   │
├──────────┼──────────┼──────────┼──────────┤
│ 2025년   │ [2026년] │ 2027년   │ 2028년   │
│ 乙巳     │ 丙午 ⭐  │ 丁未     │ 戊申     │
│ (17세)   │ (18세)   │ (19세)   │ (20세)   │
│          │ ← 현재   │          │          │
└──────────┴──────────┴──────────┴──────────┘
```

## 9. AI 사주 풀이 버튼
- "🔮 AI 사주풀이 보기" 버튼
- AI 풀이가 있으면 마크다운으로 표시
- "📥 풀이 결과 다운로드" 버튼

# ============================================================================
# 색상 테마
# ============================================================================

- 길신/좋은 것: 초록색 (st.success)
- 일반 정보: 파란색 (st.info)
- 주의사항: 주황색 (st.warning)
- 흉신/나쁜 것: 빨간색 (st.error)
- 부가 정보: 회색 (st.caption)

# ============================================================================
# 반응형 레이아웃
# ============================================================================

- 모든 섹션은 구분선(st.divider())으로 구분
- 열(columns) 사용으로 공간 효율적 배치
- 큰 데이터는 표(DataFrame) 형태로 표시
- 접을 수 있는 섹션(expander) 활용

# ============================================================================
# 사용성 개선
# ============================================================================

1. 정보 계층화: 중요한 정보(사주팔자)를 위에, 세부 정보를 아래에
2. 시각적 구분: 색상과 아이콘으로 정보의 성격 구분
3. 직관적 배치: 좌→우, 위→아래 자연스러운 흐름
4. 정보 밀도: 한 화면에 핵심 정보 모두 표시 가능
